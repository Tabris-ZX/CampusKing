# 牌堆构成

牌堆就用下面的10中牌完全组合成45张为基础牌堆

## 角色牌(5)

饭卡(meal)
攻击20,体力70,特性无

电话卡(sim)
攻击30,体力100,特性无

枪(sniper)
攻击+∞,体力50,特性 上场首回合要预瞄,无法行动

鸟女(robin)
攻击 20/50,体力 20/50,特性 双角色,两个角色都死亡才会下场

龙骑(dragon)
攻击70,体力20,特性 攻击时会额外造成已损失体力值的伤害;第一次死亡时回复至1滴血

## 技能牌(5)

圣域(sanctuary)
行动阶段可用,全场最大hp+10,每回合回复10点hp,攻击力+10

圣光(light)
当我方单位即将收到伤害时可用,完全抵御这一次的伤害

汽水(soda)
行动阶段可用,抽2,我方召唤区一名角色+40hp,对方召唤区一名角色-40hp

~~瓶中精灵(elf)
当我方召唤区有角色被击败时可用,使其回复至体力上限的1/5~~

普通一拳(bunch)
行动阶段可用,对对方全体单位造成15点伤害

伞(umbrella)
当对方打出一张技能牌时,使其无效

# 卡牌属性

## 牌属性

id:英文名
类型:角色/技能
名称:中文名
描述

# 卡属性
类型:单面卡/双面卡
a面id
b面id
(若是玩家a添加的单面卡则无b面,反之若是玩家b添加的单面卡则无a面)
