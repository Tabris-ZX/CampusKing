# 场地布置

有1个公共抽牌堆,1个公共墓地,每回合玩家抽取卡牌从公共牌库当中抽取

每人各有3个召唤区,1个手牌区

# 概念设计

**tips**: `卡`指的是双面的整张卡,`牌`指的是单面的具体对象

卡牌:

```text
双面设计,双面各1个角色牌或效果牌(一张卡的双面类型相同) 
角色牌属性有: 攻击值,体力值,特性 3项; 角色牌被击败后一般进入墓地
技能牌属性有: 效果 1项,效果分为'即时效果'和'持续效果' 效果牌被使用后一般进入墓地
```

公共抽牌堆:

```text
存放卡牌,最上面1张卡牌明置(双方都能看到)
每当抽牌堆被抽完时,将墓地所有卡牌重新洗牌,然后回归抽牌堆
每次抽牌堆洗牌时,会给予全场祝福(奇数次)或诅咒(偶数次)//todo
```

公共墓地:

```text
玩家被击败的角色卡或者使用过的效果卡均置于墓地
每当抽牌堆被抽完时,将墓地所有卡牌重新洗牌,然后回归抽牌堆
```

手牌区

```text
玩家每回合开始时从抽牌堆抽出的卡放在手牌区
玩家可以打出手牌区的卡
手牌上限是8张
```

召唤区:

```text
玩家可以放置角色牌在召唤区,1个召唤区只能放置1张角色牌
玩家可以用效果卡或者召唤区的角色牌攻击对面召唤区的角色
若被攻击方召唤区为空,则一般直接攻击玩家
```

击败奖励:
```text
当我方击败对方角色时:
若其为普通角色,则摸1张牌
若其为稀有角色,则摸1张牌,并且增加1点行动点
```

# 规则
对局开始双方先各摸两张牌,然后开始玩家HP竞标,HP大的一方先手,并且立即摸1张牌

回合制,玩家轮流行动,先手玩家首回合不能使用召唤区的角色攻击
每名玩家 100 HP,率先失去所有HP的玩家败北

## 双人

### 玩家回合

1. 回合开始: 可能会触发某些神秘效果
2. 抽卡阶段: 从抽牌堆抽取2张牌
3. 行动阶段: 1. 可以操作手牌区中的卡牌: 放置角色牌在召唤区(1回合至多放置1位)或使用技能牌 2. 使用召唤场上的角色攻击对方 3. 献祭召唤区1名角色,并且摸1张牌
4. 弃牌阶段: 若手牌数大于手牌上限则要弃至手牌上限(默认为8,会受效果影响)
5. 回合结束: 可能会触发某些神秘效果
