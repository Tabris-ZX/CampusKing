# 牌堆构成

牌堆就用下面的10中牌完全组合成45张为基础牌堆

## 角色牌(5)

饭卡(meal)
攻击20,体力100,特性无

电话卡(sim)
攻击30,体力70,特性无

枪(sniper)
攻击+∞,体力50,特性 上场首回合要预瞄,无法行动

鸟女(robin)
攻击 20/50,体力 20/50,特性 双角色,两个角色都死亡才会下场

龙骑(dragon)
攻击70,体力20,特性 攻击时会额外造成已损失体力值的伤害;第一次死亡时回复至1滴血

## 技能牌(5)

圣域(sanctuary)
行动阶段可用,全场最大hp+10,每回合回复10点hp,攻击力+10

圣光(light)
抵御对方下一次角色攻击

汽水(soda)
行动阶段可用,抽2,我方召唤区一名角色+40hp或者对方召唤区一名角色-40hp

瓶中精灵(elf)
当我方召唤区有角色被击败时可用,使其回复至体力上限的1/5

普通一拳(bunch)
行动阶段可用,对对方全体单位造成15点伤害

伞(umbrella)
抵御对方下一张技能牌

赌徒筹码(chips)
弃置至多3张牌,然后抽等量的牌数

第三只手(hands)
弃置对方一张手牌

# 卡牌属性

## 牌属性

id:英文名
类型:角色/技能
名称:中文名
攻击/体力(非角色牌留空)
效果/持续回合(非技能牌留空,若是立即生效的效果,持续回合为0,一般持续回合是2)
描述

## 卡属性
类型:单面卡/双面卡
名称:a面名称&b面名称(若是单面卡则为 '某某の牌名称')
a面id
b面id
(若是玩家a添加的单面卡则无b面,反之若是玩家b添加的单面卡则无a面)

# 祝福&诅咒

## 祝福
1. 手牌上限+1
2. 双方攻击+10
3. 每回合开始时,所有场上角色获得10点护盾,回合结束时消失
4. 抽牌数+1

## 诅咒
1. 手牌上限-1
2. 双方攻击-10
3. 每回合结束时,场上角色失去10点体力
4. 每回合攻击次数-1
